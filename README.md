# ShopFoundry

ShopFoundry helps Etsy sellers turn product ideas, images, or listing URLs into SEO-optimized Etsy listings.

## ✨ What it does

- Generate Etsy titles (SEO optimized)
- Create 13 high-performing tags
- Write conversion-focused descriptions
- Analyze listing quality score
- Suggest improvements based on Etsy SEO rules

## 🧠 Core Workflow

Input:
- Keyword / Product idea / Image / Existing listing URL

Output:
- Ready-to-paste Etsy listing content

## ⚙️ Example Prompts

### Title Generator Prompt

"Generate 3 Etsy titles under 140 characters:
- SEO-focused
- Buyer-intent focused
- Gift-oriented"

### Tag Generator Prompt

"Generate 13 Etsy tags optimized for search volume and low competition"
